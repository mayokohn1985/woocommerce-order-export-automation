== Description ==

Manually exporting WooCommerce orders?

This plugin helps you automate your workflow.

Export orders to CSV with one click — and soon automatically via cron, API and webhooks.

Perfect for:

- store owners
- agencies
- developers handling integrations

Core features:

- Manual CSV export
- Simple admin interface
- Clean and lightweight

Upcoming features (PRO):

- Automatic export (cron)
- Export on order status change
- JSON / XML export
- Webhooks / API integrations
- Custom field mapping

== Installation ==

1. Upload the plugin to /wp-content/plugins/
2. Activate the plugin
3. Go to WooCommerce → WooFlow Exporter
